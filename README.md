# INICIANDO TAILWIND CSS

## Adicionando projeto

`yarn add -D tailwindcss`
`npx tailwindcss init`

## Criar arquivo output.css e colocar:

• `@tailwind base;`<br>
• `@tailwind components;`<br>
• `@tailwind utilities;`

## Rodar o comando:

`npx tailwindcss -i caminho_do_arquivo_.css -o ./dist/output.css --watch`
